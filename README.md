# Spotify Clone 🎵

A responsive Spotify-inspired music player built with modern frontend technologies.

## Features

- Dynamic playlist system
- Play / Pause functionality
- Next & Previous song controls
- Responsive mobile-friendly design
- Sidebar navigation
- Volume control
- Seekbar progress control
- Interactive UI animations
- Album-based song organization
